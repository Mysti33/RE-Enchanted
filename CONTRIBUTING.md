# Contributing to Hastia Additions

Thank you for your interest in contributing to **Hastia Additions**!  
We welcome contributions from everyone, whether it's fixing bugs, adding features, improving documentation, or suggesting new ideas.

---

## How to Contribute

### 1. Reporting Bugs
- Check the [Issues](https://github.com/Mysti33/RE-Enchanted/issues) page first to see if the bug has already been reported.  
- If it hasn't, open a **new issue** with:
  - A clear title
  - Steps to reproduce the bug
  - Minecraft version, mod version, and Forge version
  - Any crash logs or screenshots if possible

### 2. Suggesting Features
- Please open an **issue** with a detailed description of your idea.  
- Explain how it would improve the mod or enhance gameplay.  
- We appreciate constructive discussion and realistic suggestions.

### 3. Code Contributions
1. **Fork** the repository.
2. Create a **new branch** for your changes:
   ```bash
   git checkout -b feature/your-feature-name
3. Make your changes following the **code style**:
Java 17 (or the version used in the mod)
Use the MCreator Workspace
Proper formatting and naming conventions
Include comments for complex sections

4. Test your changes in Minecraft before submitting.
5. Submit a **Pull Request (PR)** to the main branch with:
Clear title and description
Explanation of why the change is needed
Screenshots or logs if relevant

### 4. Documentation Contributions
Documentation, tutorials, and wiki updates are very welcome.
Ensure clarity and proper formatting.
Markdown formatting is preferred for GitHub wiki or README updates.

### Code of Conduct
Please follow the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
Be respectful and constructive
Avoid offensive language or behavior
Help maintain a welcoming environment for everyone
